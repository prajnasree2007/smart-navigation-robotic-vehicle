# Smart Navigation Robotic Vehicle

## Overview
The Smart Navigation Robotic Vehicle is an intelligent robotic system designed for autonomous navigation and obstacle avoidance using Arduino UNO and ultrasonic sensing technology.

The robot supports:
- Autonomous Navigation
- Bluetooth Control
- Voice Control

The system is capable of detecting obstacles, scanning surroundings, and selecting safer paths automatically for smooth movement.

## Working Principle
The ultrasonic sensor continuously detects obstacles in front of the robot.

When an obstacle is detected:
1. The robot stops.
2. The servo motor rotates the ultrasonic sensor.
3. The system scans left and right directions.
4. The controller selects the safest path.
5. The robot moves automatically.

The robotic vehicle can also be controlled through Bluetooth and Voice commands.
